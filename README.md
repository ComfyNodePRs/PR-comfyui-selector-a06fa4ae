# comfyui-selector
 
> Quick and dirty parameter generator node for ComfyUI.
>
 
> [!Note]
>  To Install, `git clone https://github.com/exdysa/comfyui-selector/` , inside your ComfyUI/custom_nodes folder, or just place the .py file inside
