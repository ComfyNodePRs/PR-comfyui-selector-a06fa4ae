# comfyui-selector
 
> Quick and dirty parameter generator node for ComfyUI.
>
 
> [!Note]
>  ```git clone https://github.com/exdysa/comfyui-selector/``` to install inside your `/ComfyUI/custom_nodes` folder, or just place the .py file inside `custom_nodes`.
